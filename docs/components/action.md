Action - 操作

## API

### Action Props

| 参数   | 说明 | 类型     | 可选值                                 | 默认值     |
|------|----|--------|-------------------------------------|---------|
| type | 类型 | string | default / primary / warning / error | primary |

### Action Group Props

| 参数   | 说明   | 类型     | 可选值 | 默认值 |
|------|------|--------|-----|-----|
| max  | 最大值  | number | -   | 2   |
| size | 间距大小 | number | -   | 8   |
