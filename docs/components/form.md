## 组件

### Attributes

| 参数   | 说明 | 类型     | 可选值                                 | 默认值     |
|------|----|--------|-------------------------------------|---------|
| type | 类型 | string | default / primary / warning / error | primary |

### Events

| 事件名            | 说明          | 参数               |
|----------------|-------------|------------------|
| onValuesChange | 数据变化时回调     | function(values) |
| onSubmit       | 提交时触发       | function(event)  |
| onFinish       | 提交并且校验通过的回调 | function(values) |
| onReset        | 重置表单回调      | function(values) |

### Methods

| 方法名            | 说明          | 参数               |

| onValuesChange | 数据变化时回调     | function(values) |

| onFinish       | 提交并且校验通过的回调 | function(values) |
