## API

### Loading Props

| 属性       | 说明       | 类型      |
|----------|----------|---------|
| isActive | 是否处于加载状态 | boolean |

### Loading Methods

| 方法   | 说明    | 类型                               |
|------|-------|----------------------------------|
| show | 激活并显示 | () => void                       |
| hide | 结束并隐藏 | (options?: LoadingProps) => void |
